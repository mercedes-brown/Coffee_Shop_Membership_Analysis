# Coffee_Shop_Membership_Analysis
Exploratory analysis of loyalty member transactions.  Includes MoM sale trends, regional performance, and market basket analysis.  Statistical tests include Apriori, chi-square, odds ratio, and Crammer's V.  Visualizations and tables included in presentation.

Data was sourced from a public [kaggle dataset]( https://www.kaggle.com/datasets/ankitrajmishra/coffee-sales-analysis-dataset?select=sales.csv ) then altered with AI to:
 1. Localize the cities and data for a US audience.
 2. Expand the number of transactions and number of items per transaction.
 3. Set pricing based on region.
 4. Update products to ones more prevalent in US coffee houses.
 5. Increase the number of customers, and adjust names.

Jupyter notebook contains most of the script used to perform this analysis, including statistical analysis.  Easy to find data, such as total revenue, units sold, and averages, may have been excluded from the final notebook.
